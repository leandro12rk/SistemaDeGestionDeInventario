System architecture

The system is organized as microservices and uses an API Gateway as a single entry point to expose one base route.

Product Service Catalog, categories, and suppliers on port 9090.
Inventory Service Movements, stock, and receipts on port 9091.
Purchase Service Purchase orders and details on port 9092.
API Gateway Single entry point on port 9000 with /api/v01.
  • Product Service on port 9090.
  • Inventory Service on port 9091.
  • Purchase Service on port 9092.
  • Gateway on port 9000 with base route /api/v01.
